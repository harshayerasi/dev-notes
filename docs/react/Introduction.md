2 options to create a new react project  

### Using npm
```bash
npm install create-react-app
create-react-app myapp
```

### Using npx
```bash
npx create-react-app myapp
```

### Install Tachyons
```bash
npm install tachyons
```

#### Importing tachyons
```javascript
import 'tachyons';
```
